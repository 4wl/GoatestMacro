# Goat

Fabric mod project for Minecraft 1.21.11.

## Requirements

- Java 21
- Gradle wrapper included in this repository

## Development

```powershell
.\gradlew.bat build
.\gradlew.bat runClient
```

In-game, run:

```text
/goat
```

The command replies with `Goat mod is loaded.` when the mod is active.
